# 영업지원 신청 시스템 — 설정 가이드 (v4)

**v4 변경점 (이전 버전 대비)**
- 기본정보: 요청자 성명/소속/연락처, 미팅목적, 미팅일정, 미팅시간, 참석인원만 남김 (이용임원/VIP 필드 제거)
- **참석자 정보를 한 곳에서 관리**: 구분(생산기술원/고객사), 회사명(생산기술원 선택 시 자동으로 "LG전자"), 이름, 소속, 직책 — 행 추가 가능, **엑셀에서 복사한 내용을 표에 그대로 붙여넣기(Ctrl+V) 가능**
- 각 예약(남산 회의실/식사, VIP식당)에서는 **기본적으로 전체 참석자가 선택된 상태**로 시작하고, 체크 해제로 제외하거나 추가로 체크할 수 있음. 고객사/사내 참석자 각각 **선택 인원수가 옆에 표시**됨
- **신청서 모달 안에서 "+ 참석자 추가"로 그 신청에만 추가로 오는 사람을 바로 등록** 가능 (식사에만 합류하는 사람, 배차에만 타는 사람 등). 등록하면 상단 "참석자 정보" 표에도 함께 반영됨
- 체크박스 목록은 좌측 정렬된 칩 형태로 컴팩트하게 표시
- **예약 추가 시 일자는 기본정보의 미팅일정을 기본값으로 자동 반영** (필요시 수정 가능)
- **남산 회의실 예약**: 목적/세팅방식/차량배차 질문 제거, 영상장비 필요여부는 유지
- **VIP식당**: 회의실 예약과 동일 구조 + 메뉴/특이사항, 접대 대상 정보 직접입력 칸 추가
- **차량 배차 + 헬기 신청을 하나로 통합** ("차량/헬기 배차") — 구간마다 출발지/목적지/방식(차량 or 헬기)/시간/탑승 인원(참석자에서 선택, 기본 전체 탑승)을 지정해서 여러 구간을 한 번에 추가
- 모든 일정에 있던 공통 "차량배차 필요여부" 질문 제거 (이제 차량/헬기 배차 카드에서 직접 신청)
- **날짜는 "2026년 01월 01일(월)" 형식으로 표시**, 텍스트로 직접 입력(숫자 8자리만 입력해도 자동 인식) 가능 + 달력 아이콘(📅)으로 선택도 가능, 요일은 자동 계산
- **시간은 자유 텍스트 입력** (예: "16:00", "16:30")
- 신청서 등록 버튼명을 **"이 예약 등록하기"** 로 변경
- **"전체 일정 제출하기" 클릭 시, 구글시트 저장과 함께 bugyeong.choi@lge.com 으로 알림 메일이 자동 발송**됩니다 (제목: "생산기술원 영업지원 신청", 본문: 구글시트 URL)

**사용할 파일은 `index_v4.html` + `Code_v4.gs` 입니다.**

## 사용 흐름

1. **기본 정보** 입력 (요청자, 미팅목적, 미팅일정/시간, 참석인원)
2. **참석자 정보** 표에 인원 등록
   - 직접 입력하거나, 엑셀에서 이름/직책 열을 복사해서 표의 첫 셀을 클릭한 뒤 Ctrl+V로 붙여넣기 (행이 부족하면 자동으로 늘어남)
   - 고객사 인원은 "구분"을 고객사로 바꾸고 회사명을 입력
3. **예약 추가**에서 필요한 항목(회의실/식사/VIP식당/차량·헬기) 클릭
   - 회의실/식사/VIP식당: 날짜·인원·시간 입력 후, 참석자 정보에서 등록된 사람 중 고객사/사내 참석자를 체크박스로 선택 (기본값은 전체 참석)
   - 이 신청에만 추가로 오는 사람이 있으면 모달 상단 "+ 참석자 추가"에서 바로 등록 (참석자 정보 표에도 함께 반영됨)
   - 차량/헬기: 날짜 입력 후 "구간 추가"로 구간별 출발지/목적지/방식(차량 or 헬기)/시간을 넣고, 그 구간에 탈 사람을 참석자 목록에서 체크 (기본값은 전체 탑승)
   - 입력을 마치면 **"이 예약 등록하기"** 클릭
4. 추가된 일정은 타임라인에 쌓이고, **"전체 일정 제출하기"**로 한 번에 제출 → 구글시트에 같은 그룹ID로 저장되고, bugyeong.choi@lge.com 으로 알림 메일이 발송됩니다

## 날짜/시간 입력 방법

- 날짜 입력칸에 숫자만 8자리 입력해도 자동 인식됩니다 (예: `20260101` → `2026년 01월 01일(월)`)
- 또는 칸 옆 📅 버튼을 눌러 달력에서 선택하면 자동으로 같은 형식으로 표시됩니다
- 시간은 자유 텍스트로 입력합니다 (예: `16:00`)
